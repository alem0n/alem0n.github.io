X-XSS-Protection

浏览器为了防护反射型xss

X-XSS-Protection:0 //不会生效

X-XSS-Protection:1 //会自动去除掉,浏览器认为有问题的部分,浏览器默认状态

X-XSS-Protection:1;mode=block //如果有恶意内容,会自动阻止页面渲染

默认容易带来安全的副作用

  • /?xxx= 通过在url处加上waf代码,使得浏览器误认为是waf是客户端发送的数据,不予执行

  • 搞安全要懂开发
  • 多读文档

渗透测试,周期短,目标是拿到shell,各种不同方式,渗透测试以目标为导向,安全研究以研究为导向,提出新的攻击方法,带动行业发展,安全研究偏向学术圈,渗透测试偏向工业圈,渗透测试不需要了解原理,安全研究需要关注动态
