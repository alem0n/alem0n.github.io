## C语言位段，位运算应用
简单的c语言c位段记录
<!-- more -->

```C
#include <stdio.h>

void prtBin(unsigned int number);

struct U0 {
  unsigned int leading : 3;
  unsigned int FLAG1 : 1;
  unsigned int FLAG2 : 1;
  int trailing : 27;
};
//位段，leading占3位，FLAG1占1位，以此类推

int main(int argc, char const *argv[]) {
  struct U0 uu;
  uu.leading = 2;
  uu.FLAG1 = 0;
  uu.FLAG2 = 1;
  uu.trailing = 0;
  printf("sizeof(uu)  = %lu\n", sizeof(uu));
  printf("&uu         = %p\n", &uu);
  //取uu的地址
  printf("(int *)&uu  = %p\n", (int *)&uu);
  //将uu地址强转为int *
  printf("*(int *)&uu = %u\n", *(int *)&uu);
  //取强制转换后指针的值
  prtBin(*(int *)&uu);
  return 0;
}

void prtBin(unsigned int number) {
  unsigned mask = 1u << 31;
  //无符号整型1u左移31位10000000000000000000000000000000
  for (; mask; mask >>= 1)
  //每一次右移一位，和number进行与运算得到相应位数的1或0，
  {
    printf("%d", number & mask ? 1 : 0); //取出对应位的1或0
  }
  printf("\n");
}
```

位段：

可以直接用位段成员名称来访问

比位移，与，或还方便

编译器会安排其中的位的排列，不具有可移植性

当所需的位超过一个int时会采用多个int

一般用于比较底层的操作硬件的场所
